# Shooting Game

## Overview
This project is a simple shooting game built with HTML, CSS, and JavaScript. The game features a player that can move and shoot bullets at enemies, with a scoring system to track the player's performance.

## Project Structure
```
shooting-game
├── src
│   ├── index.html        # HTML structure for the game
│   ├── css
│   │   └── style.css     # CSS styles for the game
│   └── js
│       ├── main.js       # Entry point for the game
│       ├── game.js       # Main game logic
│       └── entities.js    # Game entities definitions
├── package.json          # npm configuration file
├── .gitignore            # Files to ignore in version control
└── README.md             # Project documentation
```

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, etc.)
- A code editor (optional, for modifications)

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd shooting-game
   ```

### Running the Game
1. Open `src/index.html` in your web browser.
2. Use the arrow keys to move the player and the spacebar to shoot.

## Features
- Player movement and shooting mechanics
- Enemy spawning and collision detection
- Score tracking
